# LoopGo

Turli xil soha vakillarining booking ilovasi. Bu loyiha telegram bot yordamida ishga tushadi. va undan fyodalanish faqatgina telgegram bot ichida, Tashrif buyuruvchilar o'ziga kerakli xizmat turini tanlab navbatga yoziladi

## Roles

- Foydalanuvchilar `(users)` - Ilovadan ro'yxatdan o'tmasdan turib booking qila oladigan shaxslar
- Biznes egalari `(business)` - Ilovadan ro'xyatdan o'tadi va O'z biznesining ishlash vaqtini ko'rsatadi
